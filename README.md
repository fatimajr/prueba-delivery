# prueba-delivery
## Pedido de un delivery de comida a través de una llamada telefónica.
1. Inicio
2. Proceso: Coger el celular y llamar al restaurante
3. Condición: ¿Contesta el teléfono?
4. No: Regresar al paso 2
5. Sí: Realizar pedido
6. Sí: Registrar pedido y datos
7. Sí: Confirmar pedido
8. Fin
